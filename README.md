# ResumIA-compartir
Instrucciones para usar ResumIA:

Abrir la terminal y ejecutar el comando “python app.py”, estando dentro del directorio en el que se encuentra el archivo app.py.
Una vez  el servidor esté en funcionamiento, abre el enlace “127.0.0.1” que aparece en la terminal, debería de abrir la página web en tu navegador web predeterminado.
En la parte inferior de la página encontrarás el botón “Elegir archivo”. Haz clic en el botón y selecciona un archivo de audio de tu computador.
Una vez aparezca correctamente el nombre del archivo en la parte inferior de la pantalla, haz clic en “Enviar” y espera mientras el sistema procesa el audio.
Una vez finalice el procesamiento, el resumen generado se mostrará automáticamente en pantalla.
Puedes generar nuevos resúmenes utilizando el botón “crear resúmen” ubicado en la barra lateral izquierda.
Desde el menú “Ajustes” ubicado en la parte inferior de la barra lateral izquierda, puedes cambiar el tamaño de la letra y eliminar resúmenes previos.
